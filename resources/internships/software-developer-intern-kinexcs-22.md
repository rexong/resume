# Software Developer Intern | Kinexcs

Duration: Jun 2022 – Aug 2022
**Project:** Musculoskeletal Recovery Web Application
Focused on enhancing the frontend user experience and feature set of a clinical assessment tool designed for patients with musculoskeletal injuries.

* **Frontend Development:** Built and maintained responsive user interfaces using **React.js**, ensuring a seamless experience for both clinicians and patients.
* **User Flow Optimization:** Refined the application’s navigation and user journeys based on feedback from prospective customers to improve clinical usability.
* **Bug Resolution & Maintenance:** Identified and resolved critical **UI/UX bugs**, improving application stability and performance during the early stages of product launch.
* **Impact:** Delivered key feature improvements that were instrumental in securing **3 pilot partnerships** with medical clinics and institutions to test the application in real-world environments.

## Lessons Learned

* **Stakeholder-Driven Development:** Learned that fixing bugs and implementing features based on prospective customer feedback is the fastest way to achieve product-market fit and secure early partnerships.
